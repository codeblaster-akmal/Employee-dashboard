### Installation

1. Clone the repo.

   ```sh
   git clone https://github.com/codeblaster-akmal/practice-reactjs.git
   ```

2. Install NPM packages

   ```sh
   npm install
   ```

3. To start mock-server

```sh
npm install -g json-server
```

4. To run mock server

```sh
json-server --watch db.json --port 4000
```

4. Run project
   ```sh
   npm start
   ```
